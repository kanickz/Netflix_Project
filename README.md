# Netflix_Project
Netflix project using python, pandas,seaborn,matplotlib

## Overview

This project analyzes the Netflix dataset to explore trends in movies and TV shows.

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib

## Dataset

Netflix Movies and TV Shows Dataset

## Features

* Data cleaning
* Exploratory Data Analysis (EDA)
* Visualization of content trends
* Analysis of genres, ratings, and release years
